默认X轴是主轴，Y轴是侧轴

以下操作方式需写在父盒里面，即上图中的 Flex container

1. flex-direction 属性#

​ flex-direction 属性用来设置主轴方向

  1. flex-direction: row; 默认主轴为x轴，默认从左到右

    ​

  2. flex-direction: row-reverse; 修改x轴子元素排列顺序

    ​

  3. flex-direction: column; 设置主轴为y轴，默认从上到下

    ​

  4. flex-direction: column-reverse; 修改y轴子元素排列顺序

    ​

2. justify-content属性#

这里默认主轴为x轴

justify-content属性用来设置主轴上子元素的排列方式

  1. justify-content: flex-start; 从头部开始排列

    ​

  2. justify-content: flex-end; 从尾部开始排列

    ​

  3. justify-content: center; 居中排列

    ​

  4. justify-content: space-around; 平分剩余空间

    ​

  5. justify-content: space-between;先两边贴边，再平分剩余空间

    ​

3. flex-wrap 属性#

就是换行用的

  1. flex-wrap默认是不换行的，如果撑满父盒子，会导致子元素宽度缩小

    ​

  2. flex-wrap: wrap; 父盒子装不下子元素的时候换行

    ​

4. align-items属性#

align-items 用来设置单行子元素在侧轴上(默认是y轴)的排列方式

  1. align-items: flex-start; 默认值 从上到下

    ​

  2. align-items: flex-end; 从下到上

    ​

  3. align-items: center; 垂直居中

    ​

  4. align-items: stretch; 拉伸，需要取消子元素的高度才能生效

    ​

5. align-content属性#

align-content 用来设置侧轴上多行子元素的排列方式 ,可以看是否换行来判断是单行还是多行，如果出现换行就是多行子元素

  1. align-content: flex-start

    ​

  2. align-content: flex-end

    ​

  3. align-content: center

    ​

  4. align-content: space-around

    ​

  5. align-content: space-between

    ​

  6. align-items: stretch

    ​

6. flex-flow属性#

flex-flow是 flex-direction 和 flex-wrap属性的复合属性

flex-flow: row wrap;
